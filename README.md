# AI Quant Builder

AI Quant Builder helps retail investors test before they trade.

Users describe an investment idea in natural language. The product direction is to convert that idea into strategy rules, run historical backtests and risk checks, and suggest improvements before the user makes a trading decision.

## Current Stage

This repository is a public introduction and landing-page placeholder. The working prototype is private because it contains local market-data, trading, and account-related workflow code.

## Product Direction

- Natural-language investment idea input
- Strategy rule generation
- Historical backtesting
- Drawdown and risk analysis
- AI-generated improvement suggestions
- Shareable strategy validation reports
